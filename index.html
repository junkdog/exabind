Exabind Web Demo

A tachyonfx tech demo. Tachyonfx is an effects and animation library for ratatui applications. This demo runs in the browser via ratzilla, using the WebGL2 backend powered by beamterm.

Source: exabind - Interactive KDE shortcuts viewer with animated keyboard visualization.

Controls: 1-4 (toggle filters: Shift/Ctrl/Alt/Meta), ←→ (navigate categories), Esc (deselect)
